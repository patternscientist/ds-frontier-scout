# Skeptical Audit

- Why might this not actually be open? Dynamic BWT/FM-index work after 2007 may resolve important parts.
- Why might it be too saturated? String indexing is large, but this exact compact fully dynamic target is niche.
- Why might automation fail? Toy texts may not expose compressed-index lower-order terms.
- What would falsify interest? A modern linear-bit dynamic index with the target update/search times.
- Primary sources to check next: ACM version of the dynamic compressed-index paper and post-2010 dynamic FM-index/dynamic compressed suffix array literature.
