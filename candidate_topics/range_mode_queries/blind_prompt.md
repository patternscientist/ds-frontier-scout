# Blind Prompt

Given an array `A[1..n]`, build a static data structure using `O(n)` words that answers range mode queries: for any interval `[i,j]`, return an element of maximum frequency in `A[i..j]`.

Try to improve on `O(sqrt(n))` worst-case query time, or prove a lower bound for a natural class of block-decomposition or candidate-list data structures. You may use constant-time rank/select primitives as black boxes. Do not assume any literature results.
