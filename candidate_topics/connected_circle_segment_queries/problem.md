# Problem

Preprocess a connected geometric graph whose edges are straight-line segments so that a query circle reports every intersected edge output-sensitively.

The promoted residual is to determine optimal data-size/query-time tradeoffs in the connected-graph setting, and whether bottom-up construction or fractional-cascading-style reuse over nested geometric structures can improve the 2025 data structure.

Open status: explicit in the ISAAC 2025 future-work section.
