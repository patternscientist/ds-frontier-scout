# Blind Prompt

You are given a connected geometric graph with `m` straight-line edges in the plane. A query is a circle, and the data structure must report all graph edges intersecting the circle.

Use graph connectivity essentially. Seek near-linear space and output-sensitive query time, or prove a data-size/query-time lower bound specific to connected segment graphs.

As a restricted subproblem, search for connected segment arrangements where an edge-partition-tree oracle must visit many nodes even though the output is small.
