# OpenEvolve Fit

OpenEvolve suitability: 4/5.

Plausible evaluators:

- generate connected segment graphs and query circles;
- exact intersection oracle for small instances;
- AABB false-positive counter;
- partition-tree traversal simulator;
- search for high-traversal, low-output configurations.

Main caveat: finite geometry experiments do not automatically certify asymptotic lower bounds.
