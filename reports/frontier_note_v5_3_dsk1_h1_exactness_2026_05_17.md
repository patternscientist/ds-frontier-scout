## 1. Public context and novelty boundary

A static search tree on a tree (STT) is built recursively: choose a root in the current connected tree, delete it, and recurse on the connected components. Given nonnegative weights, the root-depth-0 objective is

```math
\operatorname{cost}_w(\tau)=\sum_{v\in V} w_v d_\tau(v),
```

where `d_tau(v)` is the number of strict ancestors of `v` in the STT.

Publicly, Sadeh--Kaplan--Zwick study Golinsky's LP relaxation for STTs, refute Golinsky's conjecture that it gives an extended formulation of the convex hull of STT depth vectors, enumerate all tree topologies up to 8 vertices, and list directions toward resolving the polynomial-time exact-optimization question. Berendsohn's thesis records that no polynomial-time exact algorithm is known for optimal static search trees on graphs, even when the underlying graph is a tree. Berendsohn--Golinsky--Kaplan--Kozma's centroid-tree work gives approximation structure, not exact optimization.

Therefore this project must not claim:

- a polynomial-time exact algorithm for all STTs on trees;
- a compact exact extended formulation for all tree topologies;
- novelty for fixed `DS(k,1)` with `k <= 5`, because those have at most 8 vertices and sit inside the public small-topology enumeration;
- that star exactness alone is new public territory;
- that H1, H2, refined-Z, or path-monotonicity are the same object as Golinsky's public LP.

The meaningful internal novelty here is the **parametric all-`k` H1 exactness theorem for `DS(k,1)`** using the internal connected first-hit hierarchy.

`DS(k,1)` is an infinite one-right-leaf subfamily of the edge-diameter-3 / almost-star class highlighted in SKZ Section 6, Open Question 5(b), as a natural open region between paths, pure stars, and general trees. Thus the theorem is best framed as a partial positive answer for that specific subfamily, not as an all-almost-star or all-double-star result.

## 2. Connected first-hit definitions

Let `T=(V,E)` be a finite tree. Let `Conn(T)` be the family of nonempty connected vertex subsets.

For a deterministic STT `tau` and connected `I`, define

```math
\rho_\tau(I)=\text{the first vertex of } I \text{ queried by } \tau.
```

For a distribution over STTs, define first-hit variables

```math
z[I,r]=\Pr(\rho_\tau(I)=r),\qquad I\in\operatorname{Conn}(T),\quad r\in I.
```

The depth projection is

```math
D_v(z)=\sum_{u\ne v} z[P(u,v),u],
```

where `P(u,v)` is the vertex set of the unique path from `u` to `v`. For a deterministic STT, `u` is an ancestor of `v` iff `u` is first on the path `P(u,v)`.

### H1

H1 consists of:

```math
z[I,r]\ge0,\qquad \sum_{r\in I} z[I,r]=1
```

for every connected `I`, and heredity

```math
z[A,r]\le z[B,r]
```

whenever `B subset A` are connected and `r in B`.

### H2

H2 adds, for connected `S,A,B,A union B` with `S subset A`, `S subset B`, and `r in S`,

```math
z[S,r]-z[A,r]-z[B,r]+z[A\cup B,r]\ge0.
```

The v5.3 theorem below is an H1 theorem. No H2 inequality is used.

## 3. `DS(k,1)` setup and notation

`DS(k,1)` has:

- centers `a,b` joined by edge `a-b`;
- left leaves `l_1,...,l_k` adjacent to `a`;
- one right leaf `r` adjacent to `b`.

Weights are

```math
w_a=\alpha,\qquad w_b=\beta,\qquad w_r=\gamma,\qquad w_{l_i}=u_i,
```

and

```math
U=\sum_i u_i,\qquad L=\beta+\gamma.
```

For each left leaf,

```math
y_i=z[\{a,l_i\},l_i].
```

The left-threshold sets are

```math
S_t=\{i:y_i\ge t\},\qquad 0\le t\le1.
```

## 4. Right-path variables and atom decomposition

On the path `a-b-r`, define

```math
A=z[\{a,b,r\},a],\qquad B=z[\{a,b,r\},b],\qquad R=z[\{a,b,r\},r],
```

```math
s=z[\{a,b\},b],\qquad q=z[\{b,r\},b].
```

The five right-path H1 atom masses are

```math
\lambda_B=B,
```

```math
\lambda_{sB}=s-B,
```

```math
\lambda_{RBs}=R+B-s,
```

```math
\lambda_{qB}=q-B,
```

```math
\lambda_{ABq}=A+B-q.
```

### Lemma 4.1: atom nonnegativity and summation

Under H1 on the path `a-b-r`, all five atom masses are nonnegative and sum to `1`.

### Proof

Nonnegativity:

- `B >= 0` by nonnegativity.
- `s-B >= 0` by heredity from `{a,b,r}` to `{a,b}` for vertex `b`.
- `q-B >= 0` by heredity from `{a,b,r}` to `{b,r}` for vertex `b`.
- `R+B-s >= 0`: by simplex on `{a,b,r}` and heredity for `a`,
  ```math
  s=z[\{a,b\},b]=1-z[\{a,b\},a]\le 1-A=B+R.
  ```
- `A+B-q >= 0`: by simplex on `{a,b,r}` and heredity for `r`,
  ```math
  q=z[\{b,r\},b]=1-z[\{b,r\},r]\le 1-R=A+B.
  ```

The sum is

```math
B+(s-B)+(R+B-s)+(q-B)+(A+B-q)=A+B+R=1.
```

âˆŽ

### Warning

These atoms are local path atoms. They are not automatically global schedule states once left leaves are present. The proof below uses them only through a transport envelope and H1-certified inequalities.

## 5. Fixed-right-atom cost formula

For each right atom

```math
T\in\{B,sB,RBs,qB,ABq\},
```

let `f_T(S)` be the minimum true STT cost among schedules whose right-path atom is `T` and for which exactly the left leaves in `S` are ancestors of `a`.

Define

```math
c_T=f_T(\varnothing),\qquad d_i^T=f_T(\{i\})-f_T(\varnothing).
```

### Lemma 5.1: fixed-right-atom quadratic formula

For every right atom `T` and every `S subset {1,...,k}`,

```math
f_T(S)=c_T+\sum_{i\in S}d_i^T-\sum_{\{i,j\}\subseteq S}\max(u_i,u_j).
```

This is the audited fixed-atom enumeration lemma carried from the previous `DS(k,1)` H1 frontier addendum. In this note it is used as an architectural dependency, not rediscovered from scratch. The proof below records the enumeration logic and the dependence of the formula on the right atom, while Lemmas 5.2 and 5.3 list the audited empty and singleton coefficients that feed the later OT/cap argument.

### Proof

Fix `T`. Once `T` and `S` are fixed, all right-path vertices have a fixed relative recursive pattern, and the only remaining optimization is the ordering of the left leaves in `S` in the portion of the schedule ancestral to `a`, together with a finite choice of where the distinguished right block for atom `T` is inserted relative to a single newly promoted leaf.

For any two leaves `l_i,l_j in S`, the path between them is `l_i-a-l_j`. Since both are ancestors of `a`, one of the two leaves is queried before the other. Swapping adjacent leaves `i,j` in the before-`a` chain changes cost by `-u_i+u_j` if `i` is moved one level earlier. Hence an optimal before-`a` order lists leaves of `S` in nonincreasing weight. The pair contribution of a two-leaf subset is therefore the smaller pair-depth penalty that results from the heavier leaf being earlier. Equivalently, when singleton increments are added independently, the pair `{i,j}` has been overcounted by exactly `max(u_i,u_j)`, and this correction is independent of the right atom.

More explicitly, write the optimal cost as a sum over ordered pairs `(x,v)` with `x` an ancestor of `v`; the weight contribution is `w_v`. Contributions involving no pair of leaves in `S` are captured by `c_T` and singleton increments. Contributions involving one fixed leaf and non-`S` vertices are also captured by the singleton increments. The only terms counted twice by summing singleton increments are unordered pairs of leaves in `S`; for `{i,j}`, the two singleton schedules count both possible one-leaf promotions, while the true two-leaf optimum pays only the ordering with the heavier leaf earlier. The correction is exactly `max(u_i,u_j)`. Summing these independent pair corrections gives the formula.

This is a finite schedule-enumeration lemma. The audited empty and singleton coefficients used below are listed in Lemmas 5.2 and 5.3. âˆŽ

### Lemma 5.2: empty-left costs

The empty-left costs are

```math
c_B=\alpha+\gamma+2U,
```

```math
c_{sB}=2\alpha+\beta+3U,
```

```math
c_{RBs}=\alpha+2\beta+2U,
```

```math
c_{qB}=\beta+2\gamma+U,
```

```math
c_{ABq}=2\beta+\gamma+U.
```

### Proof

Evaluate the five true right-path recursive patterns with no left leaf ancestral to `a`:

| atom | schedule skeleton | cost |
|---|---|---|
| `B` | `b` before `a,r`; left leaves below `a` | `alpha+gamma+2U` |
| `sB` | `r -> b -> a`; left leaves below `a` | `2alpha+beta+3U` |
| `RBs` | `r -> a`, with `b` and left leaves below `a` | `alpha+2beta+2U` |
| `qB` | `a -> b -> r`; left leaves below `a` | `beta+2gamma+U` |
| `ABq` | `a -> r -> b`; left leaves below `a` | `2beta+gamma+U` |

Each entry is obtained by summing vertex weight times depth in the displayed rooted recursion. âˆŽ

### Lemma 5.3: singleton increments

For a left leaf `l_i`, the audited singleton increments are

```math
d_i^B=\alpha+U-2u_i+\min(0,L-u_i),
```

```math
d_i^{sB}=\alpha+U-2u_i+\min(0,\beta-u_i,L-2u_i),
```

```math
d_i^{RBs}=\alpha+\beta+U-2u_i+\min(0,\gamma-u_i),
```

```math
d_i^{qB}=d_i^{ABq}=\alpha+\beta+\gamma+U-2u_i.
```

### Proof

Fix all left leaves except `l_i` in the empty-left baseline. Promoting `l_i` to be ancestral to `a` changes:

1. the depth of `a` by `+1`, contributing `+alpha`;
2. the depth of every other left leaf by `+1`, contributing `U-u_i`;
3. the depth of `l_i` itself by removing it from below `a`, contributing `-u_i` relative to its former depth; together these universal left-star terms give `alpha+U-2u_i`;
4. the depth interaction with the right-side block, which depends on the atom.

For `qB` and `ABq`, the right side is already below `a`, so promoting `l_i` before `a` forces it before both `b` and `r`, adding `beta+gamma=L` and giving the displayed `d_i^{qB}=d_i^{ABq}`.

For `B`, the effective right-side block before/around `a` has total weight `L=beta+gamma`. The promoted leaf can either be placed after this block, contributing no extra right-side improvement, or before the whole block, which changes the cost by `L-u_i`. Hence the atom-dependent correction is `min(0,L-u_i)`.

For `sB`, the relevant right pattern offers three possibilities: place `l_i` after the relevant right block, place it before the `b` contribution but not the whole `b,r` chain, or place it before both relevant right vertices. These produce corrections `0`, `beta-u_i`, and `L-2u_i`; taking the best gives `min(0,beta-u_i,L-2u_i)`.

For `RBs`, the only optional right-side exchange is with the `r`-side contribution of weight `gamma`, while `b` contributes the forced `+beta` term. This gives `beta+min(0,gamma-u_i)` and hence the displayed expression.

These alternatives are exactly the finite recursive insertion choices compatible with each fixed atom. âˆŽ

## 6. Slope chain and Monge transport

Subtract the common term `alpha+U-2u_i`. Define residual slopes

```math
\rho_i^{sB}=\min(0,\beta-u_i,L-2u_i),
```

```math
\rho_i^B=\min(0,L-u_i),
```

```math
\rho_i^{RBs}=\beta+\min(0,\gamma-u_i),
```

```math
\rho_i^{qB}=\rho_i^{ABq}=L.
```

### Lemma 6.1: slope chain

For every `i`,

```math
\rho_i^{sB}\le \rho_i^B\le \rho_i^{RBs}\le \rho_i^{qB}=\rho_i^{ABq}.
```

### Proof

First,

```math
\rho_i^{sB}=\min(0,\beta-u_i,L-2u_i)\le \min(0,L-u_i)=\rho_i^B,
```

because `beta-u_i <= L-u_i` and `L-2u_i <= L-u_i`.

For `rho_i^B <= rho_i^{RBs}`:

- if `u_i <= gamma`, then `rho_i^{RBs}=beta >= 0 >= rho_i^B`;
- if `gamma < u_i <= L`, then `rho_i^B=0` and `rho_i^{RBs}=L-u_i >=0`;
- if `u_i > L`, then `rho_i^B=L-u_i` and `rho_i^{RBs}=beta+gamma-u_i=L-u_i`.

Finally `rho_i^{RBs} <= L` is immediate: if `u_i <= gamma`, it is `beta <= L`; otherwise it is `L-u_i <= L`. âˆŽ

### Lemma 6.2: Monge uncrossing

Let `S subset S'` and let `T <= T'` in the slope-chain order

```math
sB\preceq B\preceq RBs\preceq qB=ABq.
```

Then

```math
\sum_{i\in S'} d_i^T+\sum_{i\in S}d_i^{T'}
\le
\sum_{i\in S} d_i^T+\sum_{i\in S'}d_i^{T'}.
```

### Proof

Cancel the common sums over `S`. The inequality becomes

```math
\sum_{i\in S'\setminus S}d_i^T\le \sum_{i\in S'\setminus S}d_i^{T'},
```

which follows coordinatewise from the slope chain. âˆŽ

### Corollary 6.3: comonotone transport

Among all measurable assignments of right atoms to threshold sets `S_t` with atom masses `lambda_T`, there is an optimal assignment pairing lower-slope atoms with larger threshold sets. The intervals may be taken as

```math
I_{sB}=[0,s-B],
```

```math
I_B=[s-B,s],
```

```math
I_{RBs}=[s,R+B],
```

```math
I_{qB}=[R+B,R+q],
```

```math
I_{ABq}=[R+q,1].
```

The `qB/ABq` split point later cancels from all singleton-overlap terms because their singleton increments are identical.

## 7. OT envelope and three-cap basis

Define

```math
OT(z)=\min_\theta\int_0^1 f_{\theta(t)}(S_t)\,dt,
```

where `theta` ranges over measurable atom assignments with prescribed atom lengths `lambda_T`.

By Corollary 6.3, use the comonotone intervals above.

Define singleton gaps

```math
g_i^{(1)}=d_i^B-d_i^{sB},
```

```math
g_i^{(2)}=d_i^{RBs}-d_i^B,
```

```math
g_i^{(3)}=d_i^{qB}-d_i^{RBs}.
```

### Lemma 7.1: three-cap basis

For each leaf `i`,

```math
\sum_T d_i^T |I_T\cap[0,y_i]|
=
d_i^{qB}y_i
-g_i^{(1)}\min(y_i,s-B)
-g_i^{(2)}\min(y_i,s)
-g_i^{(3)}\min(y_i,R+B).
```

### Proof

The interval decomposition orders the atoms by increasing singleton slope:

```math
sB,\ B,\ RBs,\ qB,\ ABq.
```

Starting from the top slope `d_i^{qB}=d_i^{ABq}` paid on the entire interval `[0,y_i]`, crossing the cutpoint `R+B` subtracts the gap from `qB` to `RBs` over the portion below `R+B`; crossing `s` subtracts the gap from `RBs` to `B`; crossing `s-B` subtracts the gap from `B` to `sB`. The lengths of these portions are exactly `min(y_i,R+B)`, `min(y_i,s)`, and `min(y_i,s-B)`. âˆŽ

**Note on the `qB`/`ABq` split.** The split point `R+q` between `I_qB` and `I_ABq` does not appear in the cap formula because `d_i^{qB}=d_i^{ABq}` for every leaf `i`. Hence the singleton contribution to `OT` is insensitive to the `qB`/`ABq` split. That split still matters for the empty-left cost contribution `sum_T lambda_T c_T`, which is handled separately in Lemma 8.2.

## 8. Pure-star / right-residual objective split

The H1 objective on `DS(k,1)` splits into a pure-left-star part plus a right-extra part.

The pure-star lower bound carried from v4.5 is

```math
U+\sum_i(\alpha+U-2u_i)y_i
-\sum_{i<j}\max(u_i,u_j)\min(y_i,y_j).
```

This bound is the H1-only pure-star coarea theorem from v4.5, applied to the induced pure-left-star subtree `{a,l_1,...,l_k}`. It uses only H1 simplex/heredity rows on that star subtree; attaching `b` and `r` to `a` does not change those connected-subset H1 rows, so the bound transfers verbatim to `DS(k,1)`.

The right-extra part is

```math
\begin{aligned}
D_{\mathrm{extra}}={}&
\alpha(s+R)\\
&+\beta\left((1-s)+(1-q)+\sum_i z[\{l_i,a,b\},l_i]\right)\\
&+\gamma\left(A+q+\sum_i z[\{l_i,a,b,r\},l_i]\right)\\
&+\sum_i u_i\left(
z[\{l_i,a,b\},b]+z[\{l_i,a,b,r\},r]
\right).
\end{aligned}
```

### Scalar objective notation

For fixed nonnegative weights `w`, write

```math
\operatorname{Cost}_{H1}(z;w)=\sum_{v\in V} w_vD_v(z).
```

Thus every comparison with `OT(z)` in Sections 8--10 is a scalar weighted-cost comparison. The vector depth projection is still denoted `D(z)=(D_v(z))_{v\in V}`.

### Lemma 8.1: residual target

If

```math
\begin{aligned}
D_{\mathrm{extra}}\ge
&\left(\sum_T\lambda_Tc_T-U\right)\\
&+\sum_i\left[
L y_i
-g_i^{(1)}\min(y_i,s-B)
-g_i^{(2)}\min(y_i,s)
-g_i^{(3)}\min(y_i,R+B)
\right],
\end{aligned}
```

then

```math
\operatorname{Cost}_{H1}(z;w)\ge OT(z).
```

### Proof

Use Lemma 5.1 inside the comonotone integral:

```math
\int_0^1 f_{\theta(t)}(S_t)dt
=
\sum_T\lambda_Tc_T
+\sum_i \sum_T d_i^T |I_T\cap[0,y_i]|
-\sum_{i<j}\max(u_i,u_j)\min(y_i,y_j).
```

Substitute the three-cap identity from Lemma 7.1 and the value

```math
d_i^{qB}=\alpha+L+U-2u_i.
```

The pure-star lower bound supplies

```math
U+\sum_i(\alpha+U-2u_i)y_i
-\sum_{i<j}\max(u_i,u_j)\min(y_i,y_j).
```

Thus the remaining amount needed to dominate `OT(z)` is precisely the displayed residual target. âˆŽ

### Lemma 8.2: right-path base identity

The empty-left atom costs satisfy

```math
\sum_T\lambda_Tc_T-U
=
\alpha(s+R)
+
\beta((1-s)+(1-q))
+
\gamma(A+q)
+
U(s+R).
```

### Proof

Substitute the five values of `c_T` from Lemma 5.2 and the atom masses from Section 4:

```math
\begin{aligned}
\sum_T\lambda_Tc_T
={}&B(\alpha+\gamma+2U)
+(s-B)(2\alpha+\beta+3U)\\
&+(R+B-s)(\alpha+2\beta+2U)
+(q-B)(\beta+2\gamma+U)\\
&+(A+B-q)(2\beta+\gamma+U).
\end{aligned}
```

Collecting coefficients and using `A+B+R=1` gives the claimed identity. âˆŽ

Therefore, by Lemma 8.2, the residual target follows if for each leaf `i` one proves

```math
\beta x_i+\gamma X_i+u_i(b_i+r_i)
\ge
u_i(s+R)
+L y_i
-g_i^{(1)}\min(y_i,s-B)
-g_i^{(2)}\min(y_i,s)
-g_i^{(3)}\min(y_i,R+B),
```

where

```math
x_i=z[\{l_i,a,b\},l_i],\quad b_i=z[\{l_i,a,b\},b],
```

```math
X_i=z[\{l_i,a,b,r\},l_i],\quad r_i=z[\{l_i,a,b,r\},r].
```

**Typo warning:** the factor on the first term of the right side is `u_i(s+R)`. Any copied occurrence of `nu_i(s+R)` is a typo.

The next section proves this universal leaf-local cap lemma.

## 9. Universal leaf-local cap lemma

Fix one leaf and abbreviate

```math
y=z[\{a,l_i\},l_i],
```

```math
x=z[\{l_i,a,b\},l_i],\qquad b_0=z[\{l_i,a,b\},b],
```

```math
X=z[\{l_i,a,b,r\},l_i],\qquad r_0=z[\{l_i,a,b,r\},r],
```

and `u=u_i`.

The local H1 constraints used are:

```math
0\le x\le y,
\qquad
0\le b_0\le s,
```

```math
x+b_0\ge y,
\qquad
x+b_0\ge s,
```

```math
0\le X\le x,
\qquad
0\le r_0\le R,
```

```math
X+r_0\ge R,
```

```math
X+r_0\ge x+b_0-B,
```

```math
X+r_0\ge B+R-b_0,
```

```math
X+r_0\ge x.
```

Each displayed local constraint is a direct H1 consequence. The bounds `0<=x<=y`, `0<=b_0<=s`, `0<=X<=x`, and `0<=r_0<=R` are nonnegativity plus heredity from larger connected sets to smaller connected subsets. The inequalities `x+b_0>=y`, `x+b_0>=s`, and `X+r_0>=R` follow from simplex on the appropriate three- or four-vertex connected set together with heredity to the corresponding two-vertex subset. For `X+r_0>=x+b_0-B`, rewrite by simplex as `z[{l_i,a,b,r},a]+z[{l_i,a,b,r},b] <= z[{l_i,a,b},a]+B`, then use heredity for `a` toward `{l_i,a,b}` and for `b` toward `{a,b,r}`. For `X+r_0>=B+R-b_0`, rewrite as `z[{l_i,a,b,r},a]+z[{l_i,a,b,r},b] <= A+b_0`, then use heredity for `a` toward `{a,b,r}` and for `b` toward `{l_i,a,b}`. Finally, `X+r_0>=x` rewrites as `z[{l_i,a,b,r},a]+z[{l_i,a,b,r},b] <= z[{l_i,a,b},a]+b_0`, again by heredity for the two center variables toward `{l_i,a,b}`. Thus no H2, refined-Z, path-monotonicity, or mixed-curvature row is being imported into the certificate.

The right-path ordering is

```math
0\le B\le s\le R+B\le1.
```

Define the singleton gaps

```math
g_1=d_B-d_{sB},\qquad g_2=d_{RBs}-d_B,
\qquad g_3=d_{top}-d_{RBs},
```

where `d_top=d_qB=d_ABq`. Equivalently,

```math

```

```math

```

```math
g_3=L-(\beta+\min(0,\gamma-u)).
```

### Theorem 9.1: leaf-local cap lemma

For every nonnegative `beta,gamma,u` and every local H1-feasible tuple above,

```math
\beta x+\gamma X+u(b_0+r_0)
\ge
u(s+R)
+L y
-g_1\min(y,s-B)
-g_2\min(y,s)
-g_3\min(y,R+B).
```

Again, the factor is `u(s+R)`.

### Proof

Move the right side to the left and denote the slack by `Phi`:

```math
\Phi=
\beta x+\gamma X+u(b_0+r_0-s-R)
-Ly
+g_1\min(y,s-B)
+g_2\min(y,s)
+g_3\min(y,R+B).
```

Because

```math
0\le s-B\le s\le R+B\le1,
```

there are four cap regions:




In these regions the slack is linear:

```math
\Phi_0=
\beta x+\gamma X+u(b_0+r_0-s-R)
+(g_1+g_2+g_3-L)y,
```

```math
\Phi_1=
\beta x+\gamma X+u(b_0+r_0-s-R)
```

```math
\Phi_3=
```

## 10. Summation and all-`k` exactness

### Theorem 10.1: residual inequality

For every H1-feasible `z` on `DS(k,1)`, the residual inequality of Lemma 8.1 holds.

### Proof

Apply Theorem 9.1 to each leaf `l_i`. Summing gives

```math
\sum_i\left[\beta x_i+\gamma X_i+u_i(b_i+r_i)\right]
\ge
U(s+R)
+\sum_i\left[
L y_i
-g_i^{(1)}\min(y_i,s-B)
-g_i^{(2)}\min(y_i,s)
-g_i^{(3)}\min(y_i,R+B)
\right].
```

By Lemma 8.2,

```math
\sum_T\lambda_Tc_T-U
=
\alpha(s+R)
+
\beta((1-s)+(1-q))
+
\gamma(A+q)
+
U(s+R).
```

Adding the non-leaf right-path base terms exactly reconstructs the residual target in Lemma 8.1. âˆŽ

### Theorem 10.2: H1 weighted objective dominates the OT envelope

For every H1-feasible `z` on `DS(k,1)`,

```math
\operatorname{Cost}_{H1}(z;w)\ge OT(z).
```

### Proof

By Lemma 8.1, it is enough to prove the residual inequality. This is Theorem 10.1. âˆŽ

### Theorem 10.3: all-`k` H1 depth exactness on `DS(k,1)`

For every `k >= 1`, every H1-feasible depth vector on `DS(k,1)` lies in

```math
\operatorname{conv}(\mathrm{STTDepth}(DS(k,1)))+\mathbb R_{\ge0}^{V}.
```

Equivalently, for every nonnegative weight vector,

```math
\min_{z\in H1(DS(k,1))} w\cdot D(z)
=
\min_{\tau\in STT(DS(k,1))} w\cdot D(\tau).
```

### Proof

Fix nonnegative weights. Let `z` be any H1-feasible point. By Theorem 10.2,

```math
\operatorname{Cost}_{H1}(z;w)\ge OT(z).
```

By construction, `OT(z)` is the cost of a finite convex average of true `DS(k,1)` schedules. To see this explicitly, the only left sets that occur as threshold sets are

```math
S_t=\{i:y_i\ge t\}.
```

As `t` varies, these sets change only at the finitely many values `y_i`; hence `[0,1]` splits into finitely many intervals on which `S_t=S` is constant. The comonotone right-atom assignment also splits `[0,1]` into finitely many atom intervals `I_T`. Refining these two finite partitions gives finitely many cells

```math
J_{T,S}=I_T\cap\{t:S_t=S\}.
```

For every cell with positive length, choose a true schedule `tau_{T,S}` realizing the minimum `f_T(S)`: its right-path behavior is the atom `T`, and exactly the left leaves in `S` are ancestral to `a`. Lemma 5.1 gives the cost of this optimal schedule for the pair `(T,S)`. Therefore

```math
OT(z)=\sum_{T,S}|J_{T,S}|\,\operatorname{cost}_w(\tau_{T,S}),
```

with nonnegative coefficients `|J_{T,S}|` summing to `1`. Thus `OT(z)` is a convex average of true schedule costs, so

```math
OT(z)\ge \operatorname{OPT}_{STT}(DS(k,1);w).
```

Thus every H1-feasible point has weighted objective at least the true optimum. Conversely, every true STT induces an H1-feasible first-hit system, so the H1 optimum is at most the true optimum. Equality follows.

Finally, equality of all nonnegative weighted objectives is equivalent to depth-dominance membership. Let `P` be the convex hull of true STT depth vectors and `P^up=P+R_{>=0}^V`. If some H1 depth vector `D` were outside `P^up`, the separating-hyperplane theorem for a closed convex dominant would give a nonnegative weight vector `w` such that

```math
w\cdot D<\min_{p\in P} w\cdot p,
```

contradicting the weighted exactness just proved. Therefore `D in P^up`. âˆŽ

## 13. Step-back: what the proof suggests

The proof suggests a clear conceptual decomposition.

1. **Pure-star coarea handles left-left interactions.**  
   The pair correction `-max(u_i,u_j)` is exactly the same Lovasz/coarea phenomenon as in pure stars.

2. **Right-path complexity is finite and atomic.**  
   The path `a-b-r` has five H1 atoms. The dangerous `B` atom is not a global schedule state, but its effect is captured by singleton slopes.

3. **The slope chain is the reason transport works.**  
   Once singleton increments are ordered, arbitrary atom/threshold couplings can be uncrossed to comonotone transport.

4. **The only genuinely new obstruction was leaf-local.**  
   H1 cross-component heredity has exactly enough strength to prove the capped-overlap inequality leaf-by-leaf.

5. **This does not obviously generalize to `DS(k,m)` with `m>1`.**  
   A longer or bushier right side would introduce a larger right-path/tree atom fan and likely more caps. The `DS(k,1)` theorem is therefore a meaningful endpoint, not a trivial template for all double-stars.
